## Problem

- Implement pow(x, n), which calculates x raised to the power n (i.e., x^n).




## Constraints
- -100.0 < x < 100.0
- -2^31 <= n <= 2^31-1
- -10^4 <= xn <= 10^4

## Example

- Input: x = 2.00000, n = 10
- Output: 1024.00000

    def myPow(self, x: float, n: int) -> float:
        
        if n == 0:
            return 1

        if n < 0:
            x = 1 / x
            n = -n

        y = 1

        while n > 1:
            if n % 2 == 0:
                x = x * x
                n = n / 2
            else:
                y = x * y
                x = x * x
                n = (n - 1) / 2

        return x * y
